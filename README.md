# Universal Electricity

This is a port of the 1.5.2 version of the universal electricity mod library for 1.7.10 including some concepts from universal electricity 1.6.4.

## Mods using this library:

- [Basic Components](https://git.tilera.org/Anvilcraft/basic-components)
- [Atomic Science](https://git.tilera.org/Anvilcraft/atomicscience)
- [Assembly Line](https://git.tilera.org/Anvilcraft/assemblyline)
- [MFFS](https://git.tilera.org/Anvilcraft/mffs)
- [ICBM](https://git.tilera.org/Anvilcraft/icbm)
- [Electric Expansion](https://git.tilera.org/Anvilcraft/electric-expansion)
- [Mekanism (tilera's fork)](https://git.tilera.org/tilera/Mekanism-1.7.10-Community-Edition)
